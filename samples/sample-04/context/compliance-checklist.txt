Data Protection — Compliance Checklists
=======================================

CHECKLIST: Customer Deletion Request
------------------------------------
[ ] Log the request in the Data Requests Register within 24 hours.
[ ] Verify the identity of the requester.
[ ] Delete the personal data within 30 days of the request.
[ ] Purge personal data from backups within the same 30-day window.
[ ] Record completion and date in the Register.

CHECKLIST: Granting Data Access
-------------------------------
[ ] Submit a request via the IT helpdesk with a business justification.
[ ] DPO approves and assigns the Data Access role.
[ ] Add the user to the quarterly recertification list.

CHECKLIST: Security Incident
----------------------------
[ ] Report to the DPO within 2 hours of detection.
[ ] Log the incident in the Incident Register using the standard template.
[ ] DPO assesses whether the supervisory authority must be notified within 72 hours.
