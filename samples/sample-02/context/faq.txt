Brightdesk — Frequently Asked Questions
=======================================

Q: How do I add a teammate?
A: Go to Settings > Team > Invite, enter their email, and choose a role (Agent or Admin).

Q: How many agents can I have on the Starter plan?
A: The Starter plan includes up to 3 agents. Upgrade to Pro for unlimited agents.

Q: Can I recover an API key if I forget to copy it?
A: No. API keys are shown only once when created. If lost, revoke it and generate a new one
   in Settings > Integrations > API Keys.

Q: Where is my data stored?
A: All customer data is stored in EU-region data centres and is encrypted at rest.

Q: What is the rate limit for the API?
A: 1,000 requests per minute per API key.
