ML101 — Introduction to Machine Learning (Official Syllabus)
============================================================
Course Code: ML101 | Institution: Riverside University | Semester: Fall 2024

COURSE OVERVIEW
---------------
This course introduces the core ideas of machine learning: how systems learn from data to
make predictions or decisions. No prior ML experience is assumed.

Learning objectives:
1. Explain the three main paradigms of machine learning.
2. Describe how a neural network is structured and trained.
3. Evaluate a model using standard classification metrics.

WEEK-BY-WEEK TOPICS
-------------------
Week 1: What is machine learning? Supervised, unsupervised, reinforcement learning.
Week 2: Neural networks — layers, activation functions, training.
Week 3: Model evaluation — accuracy, precision, recall, overfitting.

ASSESSMENT
----------
60% final exam, 40% coursework. The exam focuses most heavily on Weeks 2 and 3.

RECOMMENDED READING
-------------------
"Hands-On Machine Learning" (chapters 1–4) and the weekly chapter notes provided.
