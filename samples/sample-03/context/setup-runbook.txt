Orbit Engineering — Setup Runbook
=================================

LOCAL DEVELOPMENT SETUP
-----------------------
1. Install Node 20 and pnpm.
2. Clone the main repo: github.com/orbit/app.
3. Copy .env.example to .env and get the secret values from the team 1Password vault.
4. Run `pnpm install` then `pnpm dev` to start the local server on http://localhost:3000.

ACCESS REQUESTS
---------------
Staging access: request in #orbit-eng; granted by the Tech Lead, usually same day.
Production access: request from the Engineering Manager; requires SSO (Okta) set up first.
1Password vault access: ask your Tech Lead on your first day.

FIRST PULL REQUEST
------------------
Branch naming: feature/<short-name> or fix/<short-name>.
Open a PR into `main`. At least one approval from a senior engineer is required to merge.
All CI checks (lint, test, build) must pass before merging.
Keep PRs under 400 changed lines where possible.
