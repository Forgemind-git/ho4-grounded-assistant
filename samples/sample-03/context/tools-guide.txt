Orbit Engineering — Tools Guide
===============================

PROJECT MANAGEMENT
------------------
Task tracking: Linear. The current sprint board is the "Orbit" team in Linear.
Ask the Engineering Manager to add you to the Linear workspace.

CODE AND DEPLOYMENT
-------------------
Code lives in the github.com/orbit organisation.
CI/CD runs on GitHub Actions.
Staging deploys automatically when you merge to the `staging` branch (~3 minutes).
Production deploys automatically when you merge to `main` (~8 minutes).
Rollback: type `/rollback prod` in the #deploys Slack channel.

COMMUNICATION
-------------
Primary chat: Slack. Video calls: Zoom. Shared calendar: Google Calendar ("Orbit Eng").

MONITORING AND ALERTS
---------------------
Monitoring: Grafana dashboards (link pinned in #orbit-eng).
Alerts: PagerDuty. To join the on-call rotation, message @oncall-admin in Slack.
